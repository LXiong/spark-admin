Notice

The cluster is being started and may take up to 5 minutes. You may see the cluster status is already running but please wait until you see Mesos/Ganglia/Ephemeral/Persistent are installed and on ( icon).

If you run Spark-Admin as background service using nohup, supervisord... please notice that the setup script will likely not work due to problem in shell executing. Then you need to run this command manually, remember to replace [master_dns] with the DNS of master node

ssh -t -o StrictHostKeyChecking=no -i {{identity_file}} root@[master_dns] 'spark-ec2/setup.sh'
